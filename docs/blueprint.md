# **App Name**: NourText

## Core Features:

- Image Upload: Allow users to upload images from their device.
- Arabic OCR: Employ an OCR tool optimized for recognizing and extracting Arabic text from images.
- Text Editing: Provide a text editor for users to correct and modify the extracted Arabic text.
- Export to Word: Enable users to export the edited text directly to a Word document (.docx format).

## Style Guidelines:

- Primary color: Dark green (#34A0A4) to evoke a sense of clarity and professionalism.
- Background color: Dark gray (#343a40) to ensure readability and a modern feel.
- Accent color: Teal (#008080) for buttons and highlights, drawing associations with accuracy and growth.
- Body and headline font: 'PT Sans', a sans-serif that combines a modern look and a little warmth or personality; suitable for both headlines and body text
- Use minimalist icons to represent actions like upload, edit, and export.
- Employ a clean, intuitive layout with clear visual hierarchy for easy navigation.
